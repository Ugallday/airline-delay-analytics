# Airline Delay Analytics — Frontier Airlines @ ORD

This project analyzes operational performance for Frontier Airlines (F9) at Chicago O’Hare (ORD).

## Tools
- KNIME (data cleaning & KPI logic)
- Power BI (dashboard + modeling)

## Metrics
- Late departure rate (15+ minutes)
- Cancellation rate
- Diversion rate
- Taxi-out congestion trends

## Key Insight
Taxi-out congestion strongly correlates with late departures, indicating runway and ground movement pressure as a primary delay driver.
